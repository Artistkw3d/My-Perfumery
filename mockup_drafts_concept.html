FORMULA TOOLBAR CONCEPT
How it works: Save Draft snapshots your current ingredients. You can load any previous draft to continue from that point. When you're satisfied, click "Approve as Final" to lock the formula and unlock IFRA Certificate + MSDS Report.
TECHNICAL CONCEPT (DATABASE)
New Table: formula_drafts
ColumnTypeDescription
idINTEGER PKDraft ID
formula_idINTEGER FKParent formula
draft_numberINTEGER1, 2, 3...
nameTEXT"Draft 1", custom name
notesTEXTWhat changed
is_finalBOOLEANApproved draft
created_atTIMESTAMPWhen saved
New Table: draft_ingredients
ColumnTypeDescription
idINTEGER PKRow ID
draft_idINTEGER FKParent draft
material_idINTEGER FKMaterial ref
weightREALWeight in grams
dilutionREAL0-1 fraction
diluentTEXTDiluent type
notesTEXTIngredient note
Relationship: Formula (1) --> (many) Drafts --> (many) Draft Ingredients. Current working ingredients stay in formula_ingredients. "Save Draft" copies them into draft_ingredients. "Load Draft" copies draft_ingredients back into formula_ingredients. "Approve" marks one draft as is_final=true and changes formula status to "final".
